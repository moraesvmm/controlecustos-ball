# Regras Críticas para Agentes de IA

1. **Módulo RC & Visual**: PROIBIÇÃO CRÍTICA! Sob nenhuma circunstância altere aspectos visuais (CSS/Layout) ou lógicas fundamentais relacionadas ao Módulo RC, exceto quando explicitamente solicitado e autorizado para correções pontuais de defeitos.
2. Não modifique o sistema de expansão de fotos (lightbox), a estrutura da tabela do Controle RC, ou os gráficos da tela de Dashboard, pois eles são componentes estáveis e essenciais.
3. Se precisar implementar novas funções (ex: no Módulo Preventiva), certifique-se de que não haja vazamento de CSS que quebre as visualizações do Módulo RC ou cause scroll indesejado.
